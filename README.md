# Cloud Resume Challenge
A publicly accessible resume website hosted on AWS featuring a live visitor counter and automated CI/CD.

![Project Diagram](assets/project-diagram.png)

### Project Structure
* **`cloud-resume-challenge-iac/`**: AWS Infrastructure (Terraform/SAM) & Backend API.
* **`cloud-resume-challenge-resume/`**: Frontend website assets (HTML/CSS/JS).
